# PWA Icons Setup

To complete the PWA setup, you need to add the following icon files to this directory:

## Required Icons

1. **icon-192x192.png** - 192x192 pixels
   - Used for Android home screen and app launcher
   - Should be square with rounded corners (optional)

2. **icon-512x512.png** - 512x512 pixels
   - Used for Android splash screen and app launcher
   - Should be square with rounded corners (optional)

3. **favicon.ico** - 16x16, 32x32, or 48x48 pixels
   - Browser tab icon
   - Can be a multi-size ICO file

## Apple Touch Icon (Optional)

- **apple-touch-icon.png** - 180x180 pixels
  - Used when adding to iOS home screen

## Design Guidelines

- Use the All In One Drive logo/branding
- Ensure icons are clear and recognizable at small sizes
- Use a solid background color (matching theme color: #14b8a6)
- Keep important elements centered
- Test on both light and dark backgrounds
