## üìã Naming Convention for All Days

**Format:** `[descriptive-name].mp4`

**Rules:**
- Use lowercase letters
- Use hyphens (-) instead of spaces
- Keep it short and descriptive
- Match the PDF title as closely as possible
- Remove special characters and file extensions from the original name

**Examples:**
- ‚úÖ `weather-226-to-237.mp4`
- ‚úÖ `safety-and-your-vehicle.mp4`
- ‚úÖ `safety-margins.mp4`
- ‚ùå `English weather condition 226 to 237_video_1080.mp4` (too long, has spaces)
