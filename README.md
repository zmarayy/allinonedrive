```bash

```

3. Open [http://localhost:3000](http://localhost:3000) to view it in the browser.

## Design Features

- **Sticky Glass Navbar**: Semi-transparent with backdrop blur
- **Hero Section**: Bold typography with gradient text effects
- **Feature Cards**: Glassmorphism cards with hover animations
- **Responsive Grid**: Adapts from 1 column (mobile) to 4 columns (desktop)
- **Smooth Animations**: Hover effects and transitions throughout

## Color Palette

- Primary: #7F56D9 (Purple)
- Secondary: #3B82F6 (Blue)
- Background: Soft gradient from purple-50 to indigo-100
- Text: Gray-900 for headings, Gray-600 for body text
